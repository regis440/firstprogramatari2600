    546  f56e							; there are two special memory locations, $FFFC and $FFFE
    547  f56e							; When the atari starts up, a "reset" is done (which has nothing to do with
    548  f56e							; the reset switch on the console!) When this happens, the 6502 looks at
    549  f56e							; memory location $FFFC (and by extension its neighbor $FFFD, since it's
    550  f56e							; seaching for both bytes of a full memory address)  and then goes to the
    551  f56e							; location pointed to by $FFFC/$FFFD...so our first .word Start tells DASM
    552  f56e							; to put the binary data that we labeled "Start" at the location we established
    553  f56e							; with org.  And then we do it again for $FFFE/$FFFF, which is for a special
    554  f56e							; event called a BRK which you don't have to worry about now.
